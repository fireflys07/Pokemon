# Лабораторная работа №1 — Покемоны

Реализация иерархии покемонов и атак с последующей симуляцией боя.

## Описание

В рамках лабораторной работы реализованы:
- Классы покемонов в пакете `pokemons` (наследуются от базового класса из внешней библиотеки).
- Классы атак в пакете `moves` (наследуются от `PhysicalMove`, `SpecialMove` или `StatusMove`).
- Симуляция боя между двумя командами покемонов с использованием класса `Battle` из внешнего JAR-архива.

Программа корректно запускается, не выдаёт ошибок и демонстрирует бой в консоли.
